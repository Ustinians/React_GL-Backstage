# React项目-后台管理系统
## 前台数据交互
### 创建基本目录结构

>src
>
>​	> api	ajax相关
>
>​	> assets	公共资源
>
>​	> components	非路由组件
>
>​	> config	配置
>
>​	> pages	路由组件
>
>​	> utils	工具模块
>
>​	   App.js	应用跟组件
>
>​	   index.js	入口JS文件

### 项目准备工作

* 引入`Antd`组件库

  1. 执行 `yarn add antd` 命令进行安装

  2. 在入口文件`index.js`中引入antd样式

     ```js
     // 引入Antd样式
     import "antd/dist/antd.css"
     ```

* 引入路由

  执行`yarn add react-router-dom`安装路由
